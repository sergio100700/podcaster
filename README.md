# 🎧 Podcaster App

Aplicación web desarrollada con **React + Vite** que permite explorar podcasts y consultar sus detalles.  

## 🚀 Tecnologías utilizadas
- [React](https://react.dev/)  
- [Vite](https://vitejs.dev/)  
- [TailwindCSS](https://tailwindcss.com/)  
- React Router DOM  

---
